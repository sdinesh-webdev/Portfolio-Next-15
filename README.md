#  Portfolio Performance Optimization Initiative

## 📈 Achievement  
**Lighthouse Performance Score:** `81 → 99` (**22% improvement**)  

Executed a **comprehensive Next.js 14 SSR migration and performance optimization strategy**, delivering a production-grade web application with near-perfect performance metrics.

---

## 🔧 Key Technical Implementations  

- **SSR Architecture Migration**  
  Converted client-side React app to **server-side rendering (SSR)** with proper component separation patterns.  

- **Bundle Optimization**  
  Applied **dynamic imports** and **code-splitting**, reducing initial JS payload by **65% (245KB → 89KB)**.  

- **Loading Performance**  
  Eliminated blocking delays, achieving **sub-100ms critical resource loading** with strategic image preloading.  

- **Advanced Optimization**  
  - Intersection Observer–based lazy loading  
  - CSS-driven animations  
  - Next.js `<Image />` optimization  

---

## 📊 Performance Metrics Delivered  

- **First Contentful Paint (FCP):** `2.8s → 0.8s` (**71% faster**)  
- **Total Blocking Time (TBT):** `890ms → 180ms` (**80% reduction**)  
- **Bundle Size Reduction:** `156KB` savings via intelligent code splitting
